# 프로젝트 개요
### 이 프로젝트는 출시를 목적으로 개인이 만들던 프로젝트였고 현재는 개인이 아닌 팀 프로젝트로 전환하여 제작하고 있습니다

# 커밋 메시지 규칙

| 커밋 타입   | 의미                        | 릴리즈 노트 섹션 |
| ----------- | --------------------------- | ---------------- |
| `feat:`     | 새로운 기능 추가            | ✨ 새로운 기능   |
| `modify:`   | 기능 수정                   | 🔄 기능 수정     |
| `refactor:` | 코드 리팩토링               | ♻️ 리팩토링      |
| `fix:`      | 버그 수정                   | 🐛 버그 수정     |
| `remove:`   | 삭제                        | 🗑️ 삭제          |
| `docs:`     | 문서 수정                   | 📝 문서          |
